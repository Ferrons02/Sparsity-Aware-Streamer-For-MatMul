# Sparsity-Aware-Streamer-For-MatMul

As part of my Semester Thesis, a project of the length of 3 months, I implemented, verified and synthetized the RTL for a sparsity-aware streamer to perform matrix multiplications in AI accelerators# Sparsity-Aware Streamer for Matrix Multiplication Accelerators

This project was developed as part of my Semester Thesis, a three-month research project at ETH Zurich.

The objective was to design, implement, verify, and synthesize the RTL of a sparsity-aware streamer for sparse matrix multiplication (SpMM) accelerators. The work included evaluating different dataflows and reuse strategies, developing analytical performance models, and implementing the final architecture in SystemVerilog.

The final design achieved up to a **15× speedup** and a **19× reduction in memory footprint** compared to the baseline non-sparsity-aware implementation.
